# TCS Stock Data – Live and Latest

## Project Overview

This project focuses on analyzing the stock price data of Tata Consultancy Services (TCS). The objective is to study stock price movements, identify trends, and apply basic machine learning techniques for predictive analysis.

The project was developed using Python in Jupyter Notebook, combining data preprocessing, exploratory analysis, visualization, and machine learning.

---

## Dataset Description

The dataset contains historical stock price information, including attributes such as:

* Date
* Open Price
* High Price
* Low Price
* Close Price
* Volume

Data preprocessing steps were performed to ensure consistency and prepare the data for analysis.

## How to Run

1. Clone the repository
2. Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

3. Launch Jupyter Notebook:

```bash
jupyter notebook
```
